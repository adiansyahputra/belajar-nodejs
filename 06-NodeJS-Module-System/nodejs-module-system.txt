Modules? 
Sekumpulan code yang dapat digunakan kembali, dengan antarmuka 
yang terdefinisi

Node Modules? 
Fungsionalitas yang simpel ataupun kompleks yang tersimpan di dalam 
sebuah file javascript, yang dapat kita gunakan kembali pada aplikasi 
node js.

Node Modules 
Setiap modul di dalam node js memiliki konteks-nya masing-masing, 
tidak bisa saling tercampur dengan modul lain pada lingkup global
